## 🚀 **Step-by-Step Instructions**

### **STEP 1: Prepare Your Project Folder**

**What to do:**

1. Open **File Explorer** (Windows key + E)

2. Navigate to your project:
   ```
   E:\AL_REEM
   ```

3. **Check you have these files:**
   ```
   E:\AL_REEM\
   ├── index.html          ✅ Main page
   ├── event-details.html  ✅ Events page
   ├── assets\             ✅ Assets folder
   │   ├── css\            ✅ Stylesheets
   │   ├── js\             ✅ JavaScript
   │   └── images\         ✅ Photos
   ├── server.js           ⚠️ Optional (won't be used)
   └── README.md           ⚠️ Optional
   ```

4. **Keep this File Explorer window OPEN** - we'll need it!

---

### **STEP 2: Open Netlify Drop**

**What to do:**

1. Open your web browser (Chrome, Firefox, Edge - any browser)

2. Go to this exact address:
   ```
   https://app.netlify.com/drop
   ```
   
3. You'll see a page that looks like this:
   ```
   ┌─────────────────────────────────────┐
   │                                     │
   │     [Netlify Logo]                  │
   │                                     │
   │   Drop your site folder here        │
   │                                     │
   │   ┌─────────────────────────────┐   │
   │   │                             │   │
   │   │     [Drag & Drop Zone]      │   │
   │   │                             │   │
   │   │   Drag a folder here        │   │
   │   │                             │   │
   │   └─────────────────────────────┘   │
   │                                     │
   │   Or, if you can't drag & drop,    │
   │   browse to your folder            │
   │                                     │
   └─────────────────────────────────────┘
   ```

4. **Keep this browser tab OPEN**

---

### **STEP 3: Upload Your Project**

**Method A: Drag & Drop (Easier)**

1. **Arrange your windows side-by-side:**
   - Left side: File Explorer with `E:\AL_REEM` folder
   - Right side: Browser with Netlify Drop page

2. **Select your entire AL_REEM folder:**
   - Click once on the `AL_REEM` folder
   - OR open it and select ALL files inside (Ctrl + A)

3. **Drag to Netlify:**
   - Click and HOLD on the folder
   - Drag it over to the browser
   - Drop it in the "Drop zone" area
   - You'll see it start uploading!

**Method B: Browse & Upload (If drag doesn't work)**

1. On the Netlify Drop page, click **"browse to folder"** link

2. A file picker will open

3. Navigate to `E:\AL_REEM`

4. Select the folder or all files inside

5. Click "Open" or "Upload"

---

### **STEP 4: Wait for Upload (30-60 seconds)**

**What you'll see:**

1. **Upload Progress:**
   ```
   Uploading your site...
   [████████████░░░░░░░░] 65%
   ```

2. **Processing:**
   ```
   Processing...
   Building your site...
   ```

3. **Success!**
   ```
   ✅ Your site is live!
   ```

**Don't close the browser!** Wait until you see success message.

---

### **STEP 5: Get Your Live Link**

**After upload completes, you'll see:**

```
┌────────────────────────────────────────┐
│  ✅ Site successfully deployed!        │
```

2. **Test the link:**

## 🛠️ **Troubleshooting**

### **Problem: "No index.html found"**

**Solution:**
Make sure you have `index.html` in the root folder:
```
✅ CORRECT:
AL_REEM\
├── index.html  ← Must be here!
└── assets\

❌ WRONG:
AL_REEM\
└── website\
    └── index.html  ← Too nested!
```

---

### **Problem: Images not showing**

**Solution:**
Check image paths in your HTML:
```html
✅ CORRECT:
<img src="assets/images/logo.png">

❌ WRONG:
<img src="E:\AL_REEM\assets\images\logo.png">
<img src="/assets/images/logo.png">  (leading slash)
```

---

### **Problem: CSS not loading**

**Solution:**
Check CSS link in index.html:
```html
✅ CORRECT:
<link rel="stylesheet" href="assets/css/style.css">

❌ WRONG:
<link rel="stylesheet" href="/assets/css/style.css">
```

---

### **Problem: Can't drag and drop**

**Solution:**
1. Click "browse to folder" link instead
2. Or try different browser (Chrome works best)
3. Or zip your folder first, then upload the zip

---

### **Problem: Upload stuck at 99%**

**Solution:**
1. Wait 2 minutes (might be processing)
2. Refresh page and try again
3. Check internet connection
4. Try smaller file size (compress images)
