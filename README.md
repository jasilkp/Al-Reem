# AL-REEM MANDI Restaurant Website

A modern, responsive website for AL-REEM MANDI restaurant showcasing "The Real Arabian Taste" since 2010.

## 🌟 Features

### Design & User Experience
- **Modern Design**: Professional and creative design with smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Color Scheme**: Red (#DC2626) and White (#FFFFFF) primary colors with black accents
- **Smooth Animations**: AOS (Animate On Scroll) library integration
- **Interactive Elements**: Hover effects, smooth transitions, and engaging user interactions

### Sections
1. **Hero Section**: Eye-catching hero with floating dish animation and call-to-action buttons
2. **Events & Announcements**: Latest events, special menus, and corporate catering
3. **About Us**: Restaurant story with animated statistics (13+ years, 20 outlets, 50+ dishes)
4. **Menu**: Interactive menu with category filtering (Mandi, Al Faham, Grilled, Beverages)
5. **Outlets**: Showcase of 20 locations with contact information and hours
6. **Gallery**: Image gallery with lightbox functionality
7. **Testimonials**: Customer reviews with auto-advancing slider
8. **Team**: Meet the team members with social media links
9. **Contact**: Contact form and information with validation

### Technical Features
- **Cross-browser Compatible**: Works on all modern browsers
- **SEO Optimized**: Semantic HTML structure
- **Performance Optimized**: Optimized images and code
- **Mobile First**: Responsive design approach
- **Accessibility**: ARIA labels and semantic markup

## 🎨 Customization

### Colors
The website uses CSS custom properties (variables) for easy color customization:

```css
:root {
    --primary-color: #DC2626;    /* Main red color */
    --primary-dark: #B91C1C;     /* Darker red for hover effects */
    --white: #FFFFFF;            /* White */
    --black: #111827;            /* Black text */
    --gray-100: #F3F4F6;        /* Light gray backgrounds */
    --gray-200: #E5E7EB;        /* Border colors */
    --gray-300: #D1D5DB;        /* Secondary text */
    --gray-600: #4B5563;        /* Body text */
    --gray-800: #1F2937;        /* Dark backgrounds */
}
```

### Content Updates
1. **Text Content**: Edit the HTML files to update text content
2. **Images**: Replace placeholder images in the `assets/` folder
3. **Contact Information**: Update phone numbers, addresses, and emails
4. **Menu Items**: Add/remove menu items and update prices
5. **Team Members**: Update team information and photos

### Adding New Sections
To add new sections:
1. Add HTML structure in `index.html`
2. Add CSS styles in `assets/css/style.css`
3. Add JavaScript functionality in `assets/js/script.js` if needed

## 📱 Responsive Design

The website is fully responsive with breakpoints at:
- **Desktop**: 1200px and above
- **Tablet**: 768px to 1199px
- **Mobile**: Below 768px
- **Small Mobile**: Below 480px

## 🖼️ Image Requirements

### Recommended Image Sizes
- **Logo**: 100x100px or larger (PNG with transparency preferred)
- **Hero Images**: 800x800px or larger
- **Background Images**: 1920x1080px or larger
- **Menu Images**: 600x500px
- **Gallery Images**: 600x500px
- **Team Photos**: 600x500px
- **Testimonial Photos**: 120x120px (circular crop)

### Image Formats
- **Photos**: JPG for photographs
- **Logos/Icons**: PNG for graphics with transparency
- **Scalable Graphics**: SVG for logos and icons

## 🔧 Technical Details

### Dependencies
- **Tailwind CSS**: Loaded from CDN.
- **Isotope**: For menu filtering, loaded from CDN.
- **imagesLoaded**: Used with Isotope, loaded from CDN.
- **Google Fonts**: Poppins, Playfair Display, and Cinzel, loaded from CDN.

### Browser Support

- Firefox (latest)

### Performance Features
- Lazy loading for images

- Smooth scrolling and animations
