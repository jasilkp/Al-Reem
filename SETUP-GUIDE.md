## 🌐 How to View Your Website

1. **Double-click** on `index.html` in your folder
2. **Or** right-click → "Open with" → Choose your browser
3. **Or** drag `index.html` into any web browser

4. Optional: Run a local server (recommended for features like routing)
	 - If PowerShell blocks npx commands, use the built-in Node server:
		 - Ensure Node.js is installed
		 - In a terminal opened in this folder, run:
			 - Windows PowerShell: `node .\server.js`
		 - Then open `http://localhost:5500`
